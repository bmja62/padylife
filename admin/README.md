# 🎨 Vue 3 + Vuetify 3.7.0 Padylife Project

A modern single-page application built with **Vue 3** and **Vuetify 3.7.0**, powered by **Vite** and using **Yarn** as the package manager.

---

## 🧰 Tech Stack

- ⚡ **Vue 3 (Composition API)** — the progressive JavaScript framework
- 💎 **Vuetify 3.7.0** — Material Design component framework for Vue
- 🚀 **Vite** — fast development server and build tool
- 💅 **Sass / SCSS** — styling with variables and mixins
- 🧱 **ESLint + Prettier** — for linting and consistent code style
- 🔧 **TypeScript** *(optional)* — type safety for larger apps

---

## 📦 Prerequisites

Make sure you have the following installed:

- **Node.js** `>= 18.0.0`
- **Yarn** (v1 or v3)

If you don’t have Yarn installed:

```bash
npm install -g yarn
```

# 📥 Installation

Install project dependencies:



```bash
yarn install
```

# 🏃 Development

Run the development server with hot reload:

```bash
yarn dev
```

# 🏗️ Build for Production

To generate an optimized production build:

```bash
yarn build
```
