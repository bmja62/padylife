# 🚀 Padylife Project

A modern web application built with **[Nuxt 3](https://nuxt.com)**, using **Node.js > 18** and **pnpm** for package management.

---

## 🧰 Tech Stack

- ⚙️ **Nuxt 3** — Vue 3 framework for SSR & SSG
- 💅 **TypeScript** (optional) — safer, cleaner code
- 📦 **pnpm** — fast, disk-efficient package manager
- 🧩 **Nitro** — lightweight server engine for APIs and server routes
- 🧱 **ESLint + Prettier** — for consistent and clean code style

---

## 📦 Installation

Make sure you’re using **Node.js 18 or higher** and have **pnpm** installed globally:

```bash
corepack enable
```

Then install dependencies:

```bash
pnpm install
```


# 🏃‍♂️ Development

Start the development server with hot reload:

```bash
pnpm dev
```


# 🏗️ Build for Production

To generate an optimized production build:

```bash
pnpm build
```
